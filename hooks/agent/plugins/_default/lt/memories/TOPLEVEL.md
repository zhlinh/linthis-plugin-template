## Linthis Lint Rules

- After modifying any code file, run `linthis -i <file> -c` to check for lint issues
- Fix lint issues manually by editing the code — never use `linthis --fix` or `linthis fix`
- Re-run linthis after each fix to confirm the issue is resolved
- Use separate `-i` flags for each file: `linthis -i file1 -i file2 -c`
